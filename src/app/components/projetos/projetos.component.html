CRUD Angular

É um projeto criado com o framework Angular, é um CRUD onde o usuário pode agendar uma consulta com seu médico particular; Ao inserir os seus dados pessoais, a data, a hora e os sintomas que apresenta, esta aplicação é muito intuitiva e fácil de utilizar.

API Giphy

Este é um projeto criado com Angular que busca imagens gif no site do Giphy.

O Giphy tem uma API que podemos informar alguns parâmetros e ele nos retorna a URL das imagens.

Projeto API REST

Este é um projeto criado com NodeJS, onde uma API REST é criada para uma lista de tarefas.

Aqui usei verbos HTTP GET, POST, PUT, DELETE, usando banco de dados e processando corpo de requisição em EXPRES.

Processando parâmetros de rota em EXPRES e métodos de manipulação de dados usando SEQUELIZE.
